## 🎯 관리자 개요

### 관리자 역할
T-HOLDEM 관리자는 플랫폼의 전반적인 운영과 관리를 담당합니다:

- **시스템 관리**: 플랫폼 설정, 기능 활성화/비활성화
- **사용자 관리**: 회원 가입 승인, 권한 관리, 계정 제재
- **콘텐츠 관리**: 구인공고 승인, 부적절한 내용 모니터링
- **데이터 관리**: 통계 분석, 리포트 생성, 데이터 백업
- **고객 지원**: 문의 응답, 분쟁 조정, 기술 지원

### 관리자 권한 체계
```
┌─────────────────┐
│   슈퍼 관리자    │ ← 모든 권한, 시스템 설정
├─────────────────┤
│   운영 관리자    │ ← 콘텐츠 관리, 사용자 관리
├─────────────────┤
│   CS 관리자     │ ← 고객 지원, 문의 응답
└─────────────────┘
```
